# Minimal Template Variant

Ultra-clean, minimalist design with maximum white space and refined simplicity.

## Design Philosophy

- **Maximum White Space**: Generous spacing (py-32)
- **Light Typography**: Font weight 300 throughout
- **Minimal Borders**: 1px solid with 20% opacity
- **No Decorations**: Pure content-focused design
- **Understated Elegance**: Professional without being flashy
- **Typography-Driven**: Let content speak for itself

## Key Features

- Fixed transparent header
- Light font weights (300)
- Simple border outline buttons
- 1px borders everywhere
- Opacity hover effects only
- No shadows or gradients

## Perfect For

- High-end professional services
- Sophisticated brands
- Design-conscious clients
- Modern minimalist aesthetics
