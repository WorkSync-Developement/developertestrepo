# Modern Template Variant

A clean, contemporary design variant for insurance agency websites with smooth animations and modern UI.

## Design Philosophy

- **Clean Lines**: Minimalist design with plenty of white space
- **Smooth Animations**: Subtle transitions and hover effects
- **Bold Typography**: Strong heading hierarchy using theme fonts
- **Rounded Corners**: Modern 2xl radius on cards
- **Accent Colors**: Strategic use of accent colors
- **Responsive Design**: Mobile-first approach

## Key Features

- Sticky header with backdrop blur
- Rounded 2xl corners throughout
- Gradient overlays and effects
- Backdrop blur glass-morphism
- Modern hover animations (scale-105)
- Icon-based visual hierarchy

## Perfect For

- Forward-thinking agencies
- Tech-savvy clients
- Contemporary brands
- Modern insurance businesses
