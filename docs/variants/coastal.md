# Coastal Template Variant

The original, default template design with a beach/coastal inspired aesthetic.

## Design Philosophy

The **Coastal** variant is the original template design featuring:

- **Ocean-Inspired Colors**: Coastal blues, sandy beiges, coral accents
- **Relaxed Aesthetics**: Approachable, friendly design
- **Balanced Layout**: Neither too minimal nor too bold
- **Trust Badges**: Visual trust elements (dividers with badges)
- **Card-Based Navigation**: Expandable card menu system
- **Smooth Animations**: GSAP-powered smooth transitions
- **Professional Yet Welcoming**: Insurance-appropriate but approachable

## Key Features

### Visual Characteristics
- Default color palette: Navy (#004080), Ocean (#A7D8DE), Coral (#F76C5E), Sand (#FAF3E0)
- Playfair Display headings + Inter body text
- Card-based navigation with animated dropdown
- Trust badge dividers
- Rounded corners (rounded-xl, rounded-2xl)
- Smooth hover effects and transitions

### Special Features

#### Card-Based Navigation
- Unique expandable card menu
- GSAP stagger animations
- Location cards (multi-location)
- Company & Resources cards (standard)
- Search card integration

#### Trust Badge Divider
- Option between solid divider or trust badges
- Configurable badges (Licensed, 5-Star Reviews, etc.)
- Customizable colors and spacing

## Perfect For
- Default choice for all agencies
- Coastal/beach area insurance agencies
- Balanced professional yet approachable feel
- Full-featured template with all options

## Version
v1.0.0 - Original coastal template design (default variant)
